# Reader 阅读 - 飞牛应用

将 [Reader](https://github.com/hectorqin/reader) 服务器版打包成飞牛 fnOS 应用。

## 使用

| 地址 | 说明 |
|------|------|
| `http://NAS-IP:22389/` | Web 主页面 |
| `http://NAS-IP:22389/reader3/webdav/` | WebDAV 同步地址 |

**WebDAV 登录**: 多用户模式使用注册账号，单用户模式用户名 `default` 密码 `123456`
